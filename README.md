# katbates.com

## Commands

| Command           | Action                                           |
| :-----------------| :----------------------------------------------- |
| `pnpm install`    | Installs dependencies                            |
| `pnpm start`      | Starts local dev server at `localhost:3000`      |
| `pnpm build`      | Build the production site to `./dist/`           |
| `pnpm preview`    | Preview the build locally, before deploying      |
